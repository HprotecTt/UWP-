# 安装
下载对应版本安装包后双击打开安装，其中【arm】适用于Windows Phone，【x86】适用于32位PC，【x64】适用于64位PC，【.AppxbBundle】文件包含全平台版本及运行所需依赖，可双击安装，各独立版本安装时会自行安动相关依赖。
